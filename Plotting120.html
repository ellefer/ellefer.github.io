Box Plot

Let's start with a box plot. A box plot draws a lines where the first, second, and third quartiles are and makes them into a box. So, the box represents the middle 50% of the data. The tails extend from the minimum to the first quartile and from the third quartile to the maximum. Sometimes, when possible outliers are flagged, they are plotted as points after the tails.

We use ggplot. The ggplot command sets up the axes for the plot. The argument tells ggplot what data we are plotting. aes stands for aesthetics, and this is where you describe how variables in the data are mapped to visual properties. For our first example, we just input the xxx and yyy variable. Here, we don't have an xxx variable. We are just describing how yyy, the height, is changing. So, we input x=""x = ""x="" and y=Heighty = Heighty=Height.

However, if you just run the ggplot command, you will get a blank set of axes (delete geom_boxplot() and try it!). You need to tell it what kind of plot we want. So, we add a boxplot layer to our axes using + geom_boxplot()

ggplot(data = HW, aes(x = "", y=Height)) + geom_boxplot()
