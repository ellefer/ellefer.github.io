We have two dictionaries, one for the temporary labels (tlabels) and one for the permanent labels (plabels).

Step 0: We iterate through all the temporary vertices, label the start vertex with 0 and label the rest of the vertices with infinity.

While the temporary dictionary is not empty, we repeat Step 1 and 2:

Step 1: Find the minimum temporary vertex, and move it to the permanent list. Delete this vertex from the temporary list

Step 2: For each temporary vertex, find the permanent vertices that are adjacent to it. For each of these permanent vertices, find the value of the permanent label + the length of the edge connecting the temporary vertex to the permanent label. Replace the label of the temporary vertex with the minimum of these values.
